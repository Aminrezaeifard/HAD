# HAD Innovation Catalog

## Overview

This document provides an overview of the major innovations proposed by the HAD (Human Decides. AI Designs.) methodology.

Each innovation will be further specified in dedicated technical documents as the project evolves.

## Innovation Areas

### 1. BPMN-AI

An extension of BPMN that transforms business process models into a visual prompt language for AI-assisted enterprise engineering.

### 2. AI Service

A new BPMN component that enables AI to generate and manage Web Services and APIs based on business requirements.

### 3. Agent Task

A process element executed by an AI agent instead of a traditional workflow participant.

### 4. Smart Assignment

An intelligent assignment mechanism that dynamically selects the most appropriate user, role, or organizational unit based on business context.

### 5. AI Advisor

An AI assistant running in parallel with human activities to analyze enterprise data and provide decision support.

### 6. AI Explain

An intelligent assistant that explains why specific information is required, where it will be used, and how it impacts the business process.

### 7. Living Documentation

Automatically generated and continuously updated documentation synchronized with the enterprise system.

### 8. AI Deployment Agent

An intelligent deployment assistant capable of infrastructure analysis, software installation, configuration, and deployment planning under human approval.

### 9. Cloud Core & Local Runtime

A hybrid architecture that combines shared cloud intelligence with organization-specific private knowledge.

### 10. Partner Network

An AI-assisted ecosystem that recommends and integrates external service providers such as logistics, payment, messaging, and other enterprise partners.

### 11. AI-Generated Enterprise Artifacts

Automatic generation of forms, entities, reports, dashboards, APIs, documentation, and deployment plans from BPMN-AI models.

## Guiding Principle

All innovations within HAD follow a single philosophy:

> **Human Decides. AI Designs.**
