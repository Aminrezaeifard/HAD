# Project Overview

HAD (Human Decides. AI Designs.) is a long-term research and development project focused on redefining Enterprise Engineering through Artificial Intelligence.

The project's vision is to transform BPMN into an intelligent visual prompt language that enables AI to design enterprise systems while keeping humans responsible for all business decisions.
