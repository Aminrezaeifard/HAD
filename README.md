# HAD Vision

## Human Decides. AI Designs.

HAD (Human-Decides AI-Designs) is an AI-driven enterprise engineering methodology designed to transform the way enterprise systems are analyzed, designed, deployed, and continuously improved.

Instead of replacing human decision-makers, HAD empowers them by using AI as an engineering assistant throughout the entire software lifecycle.

The long-term vision of HAD is to establish a new methodology for AI-assisted enterprise engineering based on BPMN, multi-agent systems, shared intelligence, and living documentation.
